During my internship of final year project, I worked on a cybersecurity project focused on simulating network attacks and detecting them using an Intrusion Detection System (IDS). The main objective was to understand how cyberattacks are executed and how they can be identified and analyzed in a controlled environment.
To achieve this, I set up a virtualized laboratory using two virtual machines. One machine acted as the attacker, while the other represented the victim system equipped with the Snort IDS. This architecture allowed me to safely simulate real-world attack scenarios without impacting a production environment.

✨ Core Focus

  • Implementation of a SNORT-based Intrusion Detection System (IDS) to monitor incoming and outgoing network traffic.
  • Development and testing of custom SNORT rules to detect suspicious activity.
